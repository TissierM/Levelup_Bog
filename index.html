Un premier pas dans le monde du Code

Qu'est ce que le Code?

Il s'agit d'un language information permettant à un utilisateur d'interargir avec un appareil pouvant décoder ce language. L'utilisateur peut utiliser et travailler le Code afin de forcer l'appareil à faire précisément l'action qu'il souhaite effectuer.

Il existe différent types de language de programmation. Chaque language a été crée pour remplir une fonction précise (Html et Css sont la base d'une page internet, etc.)

Ce site a pour but d'aider les personnes à découvrir les bases d'internet, les bases de différent type de Code et les liens utiles pour débuter.
