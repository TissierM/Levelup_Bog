Les formats

Il s'agit de valeurs propres à tout les langages de programmations :

  • Int (integer) : il s'agit des chiffres entiers.
  • float : les chiffres à virgules/
  • String : les textes.
  • bolean : les valeurs true ou false.
  • array : les listes
  • objet : conteneur symbolique et autonome qui contient des informations

Les TP et TE

Un thème enfant est un thème qui hérite du code et des fonctionnalités d’un autre thème, alors appelé le thème parent. Il offre la possibilité de personnaliser un thème existant sans pour autant y apporter de modifications directes. Toutes les personnalisations sont effectuées dans le thème enfant et tous les fichiers qu’il contient prennent le pas sur ceux du thème parent. Les fichiers de ce dernier ne sont pas écrasés mais bien surchargés. En utilisant un thème enfant, vos modifications sont mises à l’abri d’un écrasement par mise à jour. En effet, c’est uniquement le thème parent qui sera mis à niveau, et jamais le thème enfant.
