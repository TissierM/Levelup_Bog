le HTML

Qu'est ce que le HTML?

L’HyperText Markup Language, généralement abrégé HTML, est le langage de balisage conçu pour représenter les pages web. C'est le squellette de votre site internet. Il vous permet de mettre en forme le contenu des pages, d’inclure des ressources multimédias dont des images, des videos, des formulaires de saisie et des programmes informatiques. Il est souvent utilisé avec le langage de programmation JavaScript et des feuilles de style en cascade (CSS).

En fait, un code HTML seul vous montre une page internet très triste.

Que faire avec ?

Il s'agit du langage de base pour créer un site internet. Le HTML sert a structurer votre contenu.

Quels sont ses avantages? ses Inconvéniens?

les Avantages :

Le HTML facilite la réalisation de liens entre un serveur et un autre. On peut l'écrir avec un simple éditeur (même bloc note permet d'écrire du HTML).

les Inconvéniens :

Mais il peut être incomplet pour réaliser certaines actions et souvant le mêlange entre contenu et présentation peut être gênant.

Au final, pourquoi utiliser ce langage?

Le HTML présente à l'ordinateur une description du contenu d'un document que l'ordinateur doit interpréter. Pour cela, il utilise des balises pour marquer les différents composants de la page.

Installer HTML?

On n'a pas besoin de logiciel précis pour utiliser le HTML, créer une page .html suffit. Il faut seulement un logiciel avec lequel on peut lire et modifier le code.

Tel que :

  • Sublime Text
  • Notepad

Du coup, à quoi ressemble un code HTML ?

Voici à quoi ressemble le début d'un code HTML =>

On peut distinguer quatre composants principaux dans un code HTML :

le "DOCTYPE" (< !Doctype html >) sert à indiquer au navigateur quelles règles d'écriture obéit le code de la page HTML.

Ensuite on commence à créer la page en indiquant au navigateur quand elle débute avec < html > et quand elle finit avec .

Puis vient le head. Entre le < head > et le on appelle tout les liens dont la page aura besoin pour fonctionner. Cette partie du code est invisible à l'utilisateur.

Et enfin on a le < body >. C'est entre ces deux balises que l'on va créer la partie visible de la page. C'est la que tout les composants, les images, les vidéos et autres objets deveront être placés.

Les "boites" qui constituent les pages en HTML sont appelées des div.

il existe plusieurs types de balises avec des fonctionnements différents que l'on peut utiliser en les plaçant dans les div.

Nous avons par exemple la balise p qui permet d'écrire du texte sur notre page.

Ce sont sur ces balises qu'agit le code CSS. Mais pour cela il est important de "nommer" les Div. En effet naturellement rien ne permet à l'ordinateur de différencier une Div d'une autre. Ce ne sont que des boites présentent dans la page web. pour permettre à l'ordinateur d'appliquer tel ou tel changement sur une Div, nous lui ajoutons la balise "class=..." : ça permet de donner un nom à la div.

Exemple : < div class="bonjour">< / div >.

Il est possible d'appeler plusieurs Div "bonjour". A ce moment, on peut appliquer le même effets à plusieurs endroits d'une page. A noter qu'il existe également la balise "id". Elle permet de mettre un indentifiant à une div. Elle permet, sur plusieurs div ayant une même class d'appliquer un changement qu'il y aura que pour celle-ci.

La base d'une page html

En premier, une page Html doit être configurée dans sa partie Head.

Déjà il faut indiquer quel charset (qui permet de définir l'encodage des caractères utilisés dans la feuille de style) le navigateur doit utiliser pour afficher la page. Sur cette page j'ai utilisé la ligne de code meta charset="UTF-8". Nous avons ensuite la balise "title" qui permet de donner un nom à la page. On peut voir ce nom apparaitre dans l'onglet de la page. Viens ensuite la balise "Link"" qui permet de lier la page HTML à la page CSS. Ici la balise ressemble à ceci : link rel="stylesheet" href="../Css/style.css"

Il exite beaucoup de méta que l'on peut appeler dans une page HTML. Comme par exemple la méta pour appeler la librairie Bootstrap avec le Link : link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css" integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm" crossorigin="anonymous".

Le corps d'une page HTML

Il existe beaucoup de méta permettant de créer une page web. Nous ne vérons ici que les principales en attendant l'avancé de ma formation

Les balises h1 à h6 sont des balises "titres" qui permettent à la fois de poser un type à la page (attention ceci est différent de la méta "title"

Afin de créer des listes, on utilise la méta ul. Pour définir les éléments, on utilise li ou ol. La seule chose qui diffèrent entre les deux balises est seulement l'affichage de la liste. li affiche une liste marquée par des points, alors que ol affiche une liste marqué par des chiffres.
