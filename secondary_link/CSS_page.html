le CSS

Qu'est ce que le CSS?

Les feuilles de style en cascade, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML.

Que faire avec ?

Il y a tellement de possibilités avec le CSS que je préfère vous proposer ce LIEN où on vous montre une partie de ces possibilités avec quelques explications.

Quels sont ses avantages? ses Inconvéniens?

Voici les avantages :

  • Les pages web se chargent plus rapidement et utilisent moins de bande passante
  • Le CSS est compatible avec tous les navigateurs
  • Le CSS peut etre utilisé pour créer des pages web faciles a imprimer
  • Le CSS permet à l'utilisateur de personnaliser une page web
  • Le CSS facilite le positionnement dans les moteurs de recherche
  • Le CSS facilite la portabilité du contenu

Installer CSS?

En construction

Il y a plusieurs moyens d'appliquer du CSS sur des morceaux de HTML. Tout dépends de ce que voulez modifier et avec quelle étendue. Voulez-vous seulement modifier un mot dans un paragraphe? Le paragraphe? Tout les paragraphes de tout votre site?

Pour cela, on revient à notre histoire de balises (retournez voir la page HTML s'il le faut) telles que DIV, P, Ul, etc.

Déjà, nous allons explorer les possibilités du CSS avec quelque chose de très simple : modifier le texte d'une page.
Prenons un exemple : la police de base de votre site ne vous plait pas et vous voulez en changer. Déjà, iol vous faut une page css pour contenir toutes vos données css. Vous créez donc une page style.css et dans votre page.html vous l'appeler en ajoutant dans le head ; < link rel="stylesheet" href="style.css" >. Voila, vos deux pages sont liées entre elles et vous pouvez désormais appliquer le css sur le html que vous avez créé. Maintenant revenons à cette police qui vous déplait.
Disons qu'il s'agisse de la police de toutes vos pages HTML. Peut importe la page, elle ne vous plait pas et vous en voulez une autre, la même présente sur toutes vos pages.
Pour cela, allez sur votre page .css et taper p { }. Cette commande pour premet de selectionner la totalité des balises < p > sur votre site internet.
Mais en faite vous voulez uniquement modifier la police des balises < p > possédans la class =texte ?
Dans ce cas, allez sur votre page .css et taper .texte{ } afin de selectionner uniquement les balises ayant cette class.
Quoi? vous avez encore changer d'avis et vous ne voulez changer que la police des balises ayant id="exception"?
Et bien, cette fois pour modifier uniquement la balise avec cette id vous devez entrez dans la page .css #exception{ }. Et maintenant vous ne voulez que seul un morceau de texte, comme juste un mot soit selectionné? Pour cela vous pouvez utiliser < span > ou bien < font > pour le changer directement sur votre page html avec du code CSS.
Maintenant que vous avez enfin réussit à selectionner ce que vous vouliez, on peut passer à la suite

Avant de changer de police, vous devez savoir que de base, les ordinateurs ne supportent pas tout les types de polices. Ils ne peuvent en décrypter que certaines telles que Times New Roman, serif, sans-serif ou encore monospace. Donc vous devez soit vous plier à la sainte volonté des machines, ou les obligez à appliquer la police que vous voulez. Pour cela, allez sur Google Font. Sur ce site vous pouver choisir une police, et récupérer un link à mettre dans le head de vos pages pour que les ordinateurs ailles chercher la source de cette police, ainsi qu'un morceau de code à placer dans votre page .css.
Ce code ressemble à ceci : font-family: 'Belgrano';. Il est composé de deux parties : font-family sert à indiquer que vous voulez appliquer une police à la balise entrée plus haut, et belgrano est le nom de la police que vous voulez.

Voila comment fonctionne le CSS : vous appelez le morceau de code que vous voulez modifier et vous lui appliquer la modification que vous souhaitez.

Voyons ce que l'on peut faire d'autre avec ce texte...

Et si le texte était en gras avec la commande font-weight:bold ou encore avec font-weight:bolder?
