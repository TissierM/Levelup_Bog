le HTML

Qu'est ce que le HTML?

L’HyperText Markup Language, généralement abrégé HTML, est le langage de balisage conçu pour représenter les pages web. C'est le squellette de votre site internet. Il vous permet de mettre en forme le contenu des pages, d’inclure des ressources multimédias dont des images, des videos, des formulaires de saisie et des programmes informatiques. Il est souvent utilisé avec le langage de programmation JavaScript et des feuilles de style en cascade (CSS).


En fait, un code HTML seul vous montre une page internet très triste.


Du coup, à quoi ressemble un code HTML ?


Voici à quoi ressemble le début d'un code HTML =>


On peut distinguer quatre composants principaux dans un code HTML :

le"DOCTYPE" (< !Doctype html >) sert à indiquer au navigateur quelles règles d'écriture obéit le code de la page HTML.

Ensuite on commence à créer la page en indiquant au navigateur quand elle débute avec < html > et quand elle finit avec .

Puis vient le head. Entre le < head > et le on appelle tout les liens dont la page aura besoin pour fonctionner. Cette partie du code est invisible à l'utilisateur.

Et enfin on a le < body >. C'est entre ces deux balises que l'on va créer la partie visible de la page. C'est la que tout les composants, les images, les vidéos et autres objets deveront être placés.

Les "boites" qui constituent les pages en HTML sont appelées des Div.

il existe plusieurs types de balises avec des fonctionnements différents que l'on peut utiliser en les plaçant dans les "Div".

Nous avons par exemple la balise p qui permet d'écrire du texte sur notre page.
